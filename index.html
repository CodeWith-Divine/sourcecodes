Frequently Asked Questions

Find answers to common questions about our projects and purchasing process.

How does the "Pay What You Want" model work?

With Lemon Squeeze integration, you can choose any price you want for each project. After clicking "Get via Lemon Squeeze," you'll be redirected to a checkout page where you can set your own price. There's a minimum of $1 to cover processing fees, but you can pay more if you find value in the project.

What's included in each project download?

Each project includes: Complete source code (HTML, CSS, JavaScript), detailed documentation with setup instructions, comments explaining key code sections, assets (images, icons), and a responsive design that works on all devices. Some projects also include bonus features or alternative implementations.

Can I use these projects in my portfolio or for commercial use?

Absolutely! You're encouraged to use these projects in your portfolio to showcase your skills. For commercial use, you have full rights to modify and use the code in your projects. However, you cannot resell or redistribute the original source code as a product.

How do I get access after purchase?

Immediately after completing your purchase through Lemon Squeeze, you'll receive an email with a download link. You'll also have access to a customer portal where you can download the files anytime. The download includes a ZIP file with all project files organized and ready to use.

What if I need help or have issues with a project?

Each project includes detailed documentation to help you get started. If you encounter any issues, you can email support at help@codewithdivine.com. We typically respond within 24 hours and are committed to helping you succeed with our projects.

Do you offer refunds?

Due to the digital nature of our products and the "pay what you want" model, we don't offer refunds. However, if you're unsatisfied with a project for any reason, please contact us and we'll work with you to resolve any issues or provide additional support.
